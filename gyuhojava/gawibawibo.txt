가위바위보 V2 과제입니다.

1. 회원가입 로직 추가
--> 프로그램이 시작되면 1. 회원가입 2. 로그인 이 나오도록 하세요.

2. 1을 입력하면 회원가입 하도록 창을 띄우고, email 을 입력 받도록 하세요.

3. email 검증(대문자 하나이상, 숫자 하나이상, 길이 8 ~ 12 사이) 하시고
만약 틀린 경우엔 뭐가 틀렸는지 출력 후 재입력 받도록 하세요.. 즉 올바른 값이 입력될때까지 계속 재 입력 받습니다.

4. 올바르게 입력된 email 이라면, '축 회원가입' 출력 후 로그인 하도록 하세요.

5. 로그인이 완료되면 게임메뉴를 출력 시키세요.
5.1 / 1. 게임시작 2. 현재 전적보기 3. 끝내기

6. 사용자가 선택한 대로 나와야 합니다.
6.1 / 1을 입력하면, 게임이 시작되어야하고, 가위바위보(1,2,3) 중 하나를 입력 받은후 게임 결과를 출력시키세요.
6.2 / 결과 출력 후에는 다음처럼 메세지를 띄우세요.
(1. 게임계속 2. 전적보기 3. 종료)
1을 입력하면 계속하고, 2를 입력하면 총 전적을 다음처럼 출력시키세요
예시 : yy년 mm월 dd일 / 전적 : 승(10), 패(4), 승률(25%) 

예시처럼 메세지 출력 후 다시 위 메뉴 (1.게임으로 돌아가기 2. 게임 종료) 를 띄우고

메뉴대로 실행할 수 있도록 하세요.
